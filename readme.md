# Tezos smart contract checklist details

## Purpose
The purpose of this Tezos smart contract checklist details (TSCD) is to foster knowledge sharing on smart contract security. The details in this repository are used as additional information / examples in the [Tezos smart contract security assessment checklist](https://github.com/InferenceAG/TezosSecurityAssessmentChecklist).

## Contribution
Anyone is invited to contribute. Please use the [TSCD template](./items/TSCD-template.md), when considering proposing a new detail.

## Disclaimer
The details in this repository are updated as needed.
